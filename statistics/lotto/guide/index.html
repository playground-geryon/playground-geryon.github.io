소개

로또 통계를 시각화합니다.

가이드
회차 선택
조회할 회차를 선택합니다.
  • 회차 변경
결과: 통계가 갱신됩니다.
번호 통계
선택한 회차의 당첨번호의 통계를 표시합니다.
  • 탭 전환
결과: 번호 번호순/당첨순으로 당첨정보 표시
미출현 번호
해당 범위에서 등장하지 않은 번호만 모아봅니다.
  • 좌측 탭에서 [미출현 번호] 선택
  • 정렬 기준 선택 후 번호 확인
결과: 미출현 번호를 기준으로 조합 저장도 가능합니다.
연속 미출현
해당 범위에서 연속 미출현 횟수를 확인합니다.
  • 좌측 탭에서 [연속 미출현] 선택
결과: 번호별 연속 미출현 횟수를 확인할 수 있습니다.
내 조합
저장된 조합 목록을 열고 관리합니다.
  • 우측 상단 [내 조합] 클릭
  • 검색/정렬/즐겨찾기/메모 관리
결과: 저장한 조합을 빠르게 찾아 적용할 수 있습니다.
조합 저장
선택한 6개 번호를 저장합니다.
  • 번호 6개 선택
  • [조합 저장] 클릭
결과: 조합이 저장되고 내 조합에서 관리됩니다.
선택 초기화
현재 선택한 번호를 모두 해제합니다.
  • [선택 초기화] 클릭
결과: 선택된 번호가 모두 해제됩니다.
