api-docs.json 업로드 후 문서 확인

소개

Swagger/OpenAPI 3.0의 api-docs.json을 분석해 문서를 확인합니다.

가이드
api-docs 업로드
swagger JSON 파일을 업로드합니다.
  • api-docs.json 업로드
  • 파일명 확인
결과: Paths/Schemas 탭에서 문서를 탐색할 수 있습니다.
엔드포인트 확인
요청/응답 스키마를 확인합니다.
  • Paths 탭에서 경로 선택
  • Parameters/Request/Responses 확인
결과: 요청/응답 정의가 상세하게 표시됩니다.
HTML 다운로드
OpenAPI 문서를 단일 HTML로 내보냅니다.
  • [HTML 다운로드] 클릭
  • 파일 저장
결과: api-docs.json 기반 HTML 문서가 로컬에 저장됩니다.
