Html Viewer

원격 URL HTML 조회·DOM 스플린터

소개

HTML 코드를 직접 입력해 DOM을 탐색하고 추출 코드를 생성합니다.

가이드
HTML 입력
좌측 에디터에 HTML을 붙여넣습니다.
  • HTML 코드 입력
  • 필요 시 script/meta 제거 옵션 설정
결과: DOM 구조 목록이 생성됩니다.
DOM 탐색/검색
DOM 구조에서 원하는 요소를 찾습니다.
  • DOM 탭에서 검색어 입력
  • 목록에서 요소 선택
결과: 선택 경로와 하이라이트가 표시됩니다.
코드 생성
선택한 DOM을 기반으로 추출 코드를 생성합니다.
  • [코드 생성] 클릭
  • 결과 탭에서 텍스트/HTML 확인
결과: 추출 결과가 우측 패널에 표시됩니다.
코드 탭
Node.js/Python 코드로 추출 스크립트를 확인합니다.
  • 코드 탭으로 이동
  • Node.js 또는 Python 선택
  • 코드 복사
결과: 선택한 언어의 스크립트를 복사할 수 있습니다.
