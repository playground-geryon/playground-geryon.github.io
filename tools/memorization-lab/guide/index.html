암기장 생성기

자동 빈칸 생성 & 난이도 분석

소개

암기장을 만들고 문장형/사전형 문제를 편집·암기·퀴즈로 학습합니다.

가이드
편집 탭 - 암기장 생성
좌측에서 암기장을 만들고 문제를 관리합니다.
  • [새로 만들기]로 암기장 생성
  • 암기장 선택 후 [새 문제] 클릭
결과: 암기장/문제 리스트가 생성됩니다.
문장 문제 템플릿
문장형 문제를 만들고 빈칸을 지정합니다.
  • 문장 입력
  • 선택 영역을 빈칸으로 추가 또는 []로 암기할 단어 표시
  • 미리보기 확인
  • 문제 저장/수정
결과: 빈칸이 적용된 문장 문제가 저장됩니다.
사전 문제 템플릿
단어-뜻 카드형 문제를 구성합니다.
  • 항목 추가로 단어/뜻 입력
  • 엑셀 업로드/템플릿 다운로드 활용
  • 문제 저장/수정
결과: 사전형 문제가 저장됩니다.
암기 탭
저장한 문제를 읽으며 학습합니다.
  • 암기할 문제 선택
  • 이전/다음 버튼으로 이동
결과: 문장형은 빈칸 표시, 사전형은 표 형태로 확인됩니다.
퀴즈 탭
문제를 풀고 채점합니다.
  • 문제 선택
  • 정답 입력 후 [채점] 클릭
  • 필요 시 정답 보기/리셋
결과: 정답 수와 결과가 표시됩니다.
