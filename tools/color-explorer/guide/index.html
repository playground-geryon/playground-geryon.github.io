소개

이미지에서 팔레트를 추출하고 대비/조합을 빠르게 확인합니다.

가이드
이미지 업로드
파일 업로드 또는 링크로 이미지를 불러옵니다.
  • 이미지 업로드 버튼 클릭 또는 드래그&드롭
  • 필요 시 링크로 불러오기 사용
결과: 이미지가 캔버스에 표시되고 팔레트가 자동 생성됩니다.
스포이드 & 확대/축소
캔버스에서 색상을 픽킹하고 확대/축소로 디테일을 확인합니다.
  • 스포이드 아이콘 활성화
  • 캔버스 클릭으로 색상 선택
  • 슬라이더로 확대/축소
결과: 선택한 색상 HEX/RGBA가 표시되고 복사할 수 있습니다.
팔레트 요약/상세
유사색 묶음 요약과 전체 색상 목록을 확인합니다.
  • 요약/상세 탭 전환
결과: 상위 색상 통계와 전체 색상 분포를 확인할 수 있습니다.
